완전 초보자용 가이드 (100% 무료 구동)

🎵 다온 AI 음악 생성 스튜디오 사용법

개인용 GPU(컴퓨터 사양)가 없어도 클라우드 GPU를 활용해 나만의 고품질 음악을 만드는 방법

📋 구동을 위해 준비해야 할 것 (딱 2가지)

이 시스템은 고성능 인공지능 음악 모델을 구동하기 때문에 개인 PC 사양이 아닌 무료 클라우드 컴퓨터(Kaggle GPU)를 빌려 사용합니다. 이를 위해 아래 2가지 사이트의 무료 계정이 필요합니다.

  1. Kaggle (캐글) 계정: 인공지능 음악 모델을 구동할 고성능 그래픽카드(GPU)를 1주일에 30시간씩 무료로 대여해 주는 구글 산하 사이트입니다. (kaggle.com 회원가입)
  2. Ngrok (엔그록) 계정: 캐글 가상 컴퓨터에서 생성된 음악 서버를 우리 집 컴퓨터로 안전하게 다리 놓아 연결해 주는 터널링 서비스입니다. (ngrok.com 회원가입)

🚀 서버 구동 및 웹 UI 연결 방법

1
Ngrok 토큰(Authtoken) 복사하기
외부 연결을 안전하게 터널링하기 위해 나만의 고유 키(Token)를 가져와야 합니다.
  • Ngrok 대시보드에 가입하고 로그인합니다.
  • 왼쪽 메뉴바에서 Your Authtoken 메뉴를 클릭합니다.
  • 화면에 나타나는 영어와 숫자로 길게 된 문자열 옆의 Copy 버튼을 눌러 메모장에 따로 적어둡니다. (서버 실행 시 한 번 입력해 줄 예정입니다.)
2
Kaggle(캐글) 전화번호 인증 활성화
캐글 가상 컴퓨터의 그래픽카드(GPU) 대여 권한과 외부 통신망(Internet)을 활성화하기 위해 최초 1회 전화번호 인증이 필요합니다.
  • 캐글 로그인 후, 우측 상단 프로필 클릭 ➔ Settings(설정)로 이동합니다.
  • 설정 메뉴 중간의 Phone Verification(전화번호 인증) 항목을 찾아 휴대폰 번호 인증을 완료합니다. (완전 무료이며 로봇 악용 방지용입니다.)
⚠️ 주의: 전화번호 인증을 안 하면 서버 구동이 불가능합니다!
인증을 완료하지 않으면 다음 단계에서 그래픽카드(GPU)를 활성화하거나 인공지능 모델 다운로드(인터넷 권한)를 할 수 없어 실행 도중 에러가 발생합니다.
3
Kaggle에 노트북 업로드 및 설정 세팅
공유받으신 폴더 안에 있는 Jupyter 노트북 파일을 캐글 서버에 업로드하여 세팅합니다.
📄 업로드용 노트북 파일 위치 (ACE_Step_XL_Ngrok_Kaggle.ipynb)
  1. Kaggle 홈 좌측 메뉴에서 Create(생성) ➔ New Notebook(새 노트북)을 클릭합니다.
  2. 노트북 편집창 좌측 상단 메뉴에서 File ➔ Upload notebook을 누르고, 위 경로에 위치한 ACE_Step_XL_Ngrok_Kaggle.ipynb 파일을 넣어 업로드합니다.
  3. 업로드 후 편집창 오른쪽 설정(Settings) 사이드바에서 다음 2가지 옵션을 반드시 세팅합니다:
    • Accelerator (가속기): None으로 되어 있는 것을 GPU T4 (또는 GPU T4 x2)로 변경합니다. (AI 가속 그래픽카드 빌리기)
    • Internet on (인터넷 켜기): 스위치 토글을 켜줍니다. (인공지능 모델 파일을 외부망에서 받기 위함)
4
Ngrok 토큰 주입 및 서버 최종 구동
서버를 실행하고 연결 링크를 발급받는 단계입니다.
  1. Kaggle 화면을 아래로 쭉 스크롤하여 맨 마지막에 있는 **Cell 5 (Set Ngrok Token & Launch!)** 셀을 찾습니다.
  2. 코드 안에서 NGROK_TOKEN = "YOUR_NGROK_TOKEN_HERE" 부분의 따옴표 안을 지우고, 1단계에서 복사해 둔 개인 Ngrok 토큰을 붙여 넣습니다. 
    NGROK_TOKEN = "복사한_개인_토큰_값_여기에_입력"
  3. Kaggle 상단 툴바 메뉴에서 Run All (또는 전체 셀 실행 버튼)을 클릭합니다.
  4. 서버가 동작하면서 AI 음악 모델을 순차적으로 내려받습니다. (약 3~4분 소요)
  5. 서버 가동이 완료되면, 맨 밑 실행창 로그창에 아래와 같은 형식을 포함하는 연결 주소가 나타납니다. **주소를 마우스로 드래그해서 복사**합니다. 
    =======================================================
🚀 NGROK URL: https://xxxx-xxxx-xxxx.ngrok-free.app
=======================================================
5
웹 UI 실행 및 서버 연동하여 음악 만들기
이제 주소 세팅이 끝났습니다! 로컬 스튜디오 웹 화면을 열어 연동해 줍니다.
  1. 공유받은 폴더에 있는 index.html 파일을 마우스 더블클릭하여 인터넷 브라우저로 켭니다.
    🌐 다온 음악 스튜디오 화면 열기 (index.html)
  2. 웹 화면 맨 상단의 **API 서버 연결 설정** 입력창에 4단계에서 복사한 `https://xxxx-xxxx-xxxx.ngrok-free.app` 주소를 입력하고 **[서버 연결]** 버튼을 누릅니다.
  3. 연결 성공 시 우측에 녹색 **[연결됨]** 배지가 켜지고, 그래픽카드(VRAM) 리소스 잔량이 표시됩니다.
  4. 원하는 가사를 적고, 캡션 스타일을 선택한 뒤 맨 아래 **[AI 음악 생성 시작]** 버튼을 클릭하면 고품질 인공지능 음악이 생성됩니다!
🔒 최초 1회 경고창 우회 방법 (오디오 재생 안 됨 에러 해결)
무료 Ngrok 터널의 정책 때문에 첫 실행 시 오디오 파일이 웹 재생기에서 플레이가 차단될 수 있습니다. 해결을 위해 복사한 Ngrok 주소(`https://xxxx.ngrok-free.app`)를 인터넷 브라우저의 새 탭 주소창에 그대로 치고 들어간 뒤, 파란색 화면 중앙에 보이는 **[Visit Site]** 버튼을 **딱 한 번만 클릭**해 줍니다. 이렇게 하면 브라우저 쿠키가 등록되어 정상적으로 오디오 재생 및 다운로드가 원활하게 작동합니다.
